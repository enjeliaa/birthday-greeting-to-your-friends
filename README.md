# 🎂 Birthday Greeting

A personal, elegant birthday greeting website — built with **React JS + Vite**.  
Simple, beautiful, and made with love to celebrate the people you care about.

---

## ✨ Features

- **Elegant Hero Section** — Photo, name, and a heartfelt opening quote
- **Birthday Card** — A personal message, styled like a real letter
- **Interactive Cake** — CSS-animated candles you can "blow out" with a click
- **Background Music** — Plays automatically on first interaction, togglable anytime
- **Fully Responsive** — Looks great on mobile and desktop

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```
Then open your browser at `http://localhost:xxxx`
---

## 🌐 Deployment

This project can be deployed for free on **Vercel** — giving you a public link you can send to anyone, accessible from any device.
